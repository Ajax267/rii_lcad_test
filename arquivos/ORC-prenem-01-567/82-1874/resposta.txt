&nbsp; &nbsp; &nbsp; O progresso da civilização humana sempre andou lado a lado com o avanço do conhecimento técnico e científico. O conhecimento científico e o uso das tecnologias, mesmo as mais primitivas, sempre foram fundamentais para a sobrevivência humana. Por exemplo, os modos primitivos de saneamento básico foram fundamentais para o crescimento das cidades, tendo em vista que amenizaram questões de higiene, doenças, entre outras questões básicas para o bem-estar humano. O progresso da técnica, permitiu a criação de praticamente tudo aquilo que é hoje ferramenta fundamental em nosso cotidiano. Com a especialização e avanço do conhecimento científico é possível hoje o acesso à medicina moderna, à cidades planejadas, informação, meios de transporte, ferramentas de comunicação, e até mesmo o conhecimento do universo através da exploração espacial.&nbsp;<p>&nbsp; &nbsp; &nbsp; &nbsp;O avanço da ciência, permitiu o avanço das mais diversas tecnologias. Até mesmo as que são utilizadas como ferramenta de inclusão para deficientes, através de tecnologias assistivas. Por exemplo, leitores de telas para deficientes visuais.&nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;A ciência e a técnica são de extrema importância para o nosso desenvolvimento, mas é necessário atenção para que dentro da pesquisa científica haja sempre um compromisso com a humanidade. A prática da evolução da tecnologia, deve respeitar os valores e princípios de uma sociedade.  Os limites do que é inventado, estudado e experimentado, devem ser os limites da ética humana.</p>
